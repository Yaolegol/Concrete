# Старт проекта

## Client

### 1. Установить зависимости
> npm ci

### 2. Запустить dev сервер
> npm start

Проект доступен на **8000** порту

## Server

### 3. Перейти в директорию сервера
> cd ./server

### 4. Установить зависимости
> npm ci

### 5. Запустить dev сервер
> npm start

Сервер использует **8001** порт
