# Start project

## Client

### 1. Install dependencies
> npm ci

### 2. Start dev server
> npm start

Client dev server use **8000** port

## Server

### 3. Go to server dir
> cd ./server

### 4. Install dependencies
> npm ci

### 5. Start dev server
> npm start

Server listen **8001** port

### 6. Enjoy!
